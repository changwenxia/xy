# fe-single-service


## usage
1 创建工作目录
mkdir test
cd test
2 生成一个package.json
npm init -y
3 存放组件的文件夹
mkdir blocks
4 开启服务
node D:\前端开发\work-place\xiyun-programmes\xy\packages\cli\bin\xy.js ui 
tip: 自己复制本机上的绝对路径
5 安装 npm 包 & 下载区块

yarn install
